### About The Project

This project is created for a Requirement Based Academic Match Making Web Application. The REST API was created using undermentioned technologies.

- Java Springboot
- Hibernate ORM
- Hibernate Validator
- Object Mapper
- Lombok
- JWT tokens
- SLF4J, LogBack
- Firebase
- MySQL
